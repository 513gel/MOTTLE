About

Settings, theme, and filename persist in your browser's local storage when MOTTLE is hosted on a real site (like GitHub Pages) — including when installed to your home screen. In some sandboxed previews storage is blocked, so settings last for the session only.

MOTTLE is one HTML file. It runs the same in any modern browser — phone, tablet, PC. No uploads, no tracking, no dependencies.
